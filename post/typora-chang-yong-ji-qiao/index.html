typora快捷键😀

一到六级标题 Ctrl+1到Ctrl+6 n个#(n=1~6)
斜体 Ctrl+I *斜体* 斜体
加粗 Ctrl+B **加粗** 加粗
高亮 ==高亮== 高亮
下划线 Ctrl+U <u>下划线</u> 下划线
删除线 ~~删除~~ 删除
分割线 用三个以上的*号、-号、_来建立一个分隔线
行内代码 在两个`中插入
代码块 ```+回车键
超链接 Ctrl+K [链接名称](链接地址)或直接使用链接地址 网站
插入图片 Ctrl+Shift+I ![]()->选择图片
目录生成 [toc]
LeTex Ctrl+Shift+M

有序列表

快捷键：Ctrl+Shift+[

示例:

  1. a
    1. b
    2. c
    3. d
  2. e
  3. f

注意：

  1. 编号后，一定要有空格。
  2. 行末回车，序号会自动添加。
  • 行末按方向下键，序号不会自动添加。

  1. 添加/删除某一行内容，序号会自动调整，

  2. 降级快捷键：Tab

    升级快捷键：Shift+Tab或回车

无序列表

快捷键：Ctrl+Shift+]

示例:

  • A
  • B
    • C
    1. a
    2. b
      • c
      • d
  • C

创建方式：* A 或 - A 或+ A

注意：

  1. *或-后，一定要有空格！

  2. 降级快捷键：Tab

    升级快捷键：Shift+Tab或回车

  3. 有序列表和无序列表可以交错使用。

任务列表

创建方式：- [ ] A

注意：

  1. -, [, ]后，都一定要有空格! 括号一定是英文括号。
  2. 无快捷键，可以通过鼠标操作。
  • 选中文字->段落->任务列表

表格

快捷键：Ctrl+T

  1. 创建方式：|xx|xx|->回车。
  2. 注意：添加一行，只需按下Ctrl+回车。
  3. 其他操作：
    1. 调整表格行列值。
    2. 左中右对齐。
    3. 插入/删除表格行列。
    4. 删除表格。

注意：添加一行，

1 2 3

代码区块

快捷键：Ctrl+Shift+K

import time 
for in in range(10):
    print(i)

创建方式：```或~~~java/C++/python···

脚注

创建方式：
[^要注明的文本]
实例演示：
实例[^xx]
[^xx]:一个实例
